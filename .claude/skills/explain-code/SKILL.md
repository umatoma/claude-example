name: explain-code
description: コードを図や例文を使って説明します。コードの仕組みを説明するとき、コードベースについて教えるとき、またはユーザーが「これはどう動くの？」と質問したときに使用します。
---

コードを説明する際は、必ず以下を含めてください：

1. **分かりやすい例えから始める**: コードを日常生活のものに例える
2. **図を描く**: ASCIIアートを使ってフローや構造、関係性を示す
3. **コードを順に解説する**: 何が起きているかをステップごとに説明する
4. **ハマりポイントを指摘する**: よくある間違いや誤解は何か？

説明は会話調で分かりやすく。複雑な概念には複数の例え話を使いましょう。
