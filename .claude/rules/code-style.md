# コーディング規約

## 言語

- TypeScript を使用する（ファイル拡張子: `.ts`）

## モジュールシステム

- ESM（`import`/`export`）を使用する

## 型注釈

- 関数の引数・戻り値に型を明示する
- `any` の使用を避け、適切な型を定義する

## フォーマット

- インデント: スペース2つ
- 文字列: シングルクォートを使用する
- セミコロン: 文末に付ける

## 命名規則

- 変数・関数: キャメルケース（例: `getUserName`）
- クラス: パスカルケース（例: `UserService`）
- 定数: 大文字スネークケース（例: `MAX_RETRY_COUNT`）
- 型・インターフェース: パスカルケース（例: `UserResponse`）

## 関数スタイル

- アロー関数を優先する

## コメント

- 日本語で記述する
