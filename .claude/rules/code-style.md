# コーディング規約

## モジュールシステム

- アプリコードは CommonJS（`require`/`module.exports`）を使用する

## フォーマット

- インデント: スペース2つ
- 文字列: シングルクォートを使用する
- セミコロン: 文末に付ける

## 命名規則

- 変数・関数: キャメルケース（例: `getUserName`）
- クラス: パスカルケース（例: `UserService`）
- 定数: 大文字スネークケース（例: `MAX_RETRY_COUNT`）

## 関数スタイル

- アロー関数を優先する

## コメント

- 日本語で記述する
